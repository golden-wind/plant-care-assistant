## Android 版本

### 方法一：GitHub Actions自动构建（推荐 ⭐）

**最简单，无需安装开发环境**

```bash
# 1. 进入项目目录
cd /home/admin/workspace/plant-care-assistant

# 2. 初始化git并推送到GitHub
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/plant-care-assistant.git
git push -u origin main

# 3. 访问GitHub Actions页面，等待构建完成
# https://github.com/你的用户名/plant-care-assistant/actions

# 4. 下载APK artifact
# 文件名: app-debug.apk
```

### 方法二：本地构建（需要Java）

```bash
cd /home/admin/workspace/plant-care-assistant

# 使用一键脚本（需要安装Java JDK 17）
./build-app.sh debug

# 或手动构建
npm install
npm run build
npx cap sync android
cd android && ./gradlew assembleDebug

# APK位置: android/app/build/outputs/apk/debug/app-debug.apk
```

### 方法三：使用Android Studio

1. 安装 [Android Studio](https://developer.android.com/studio)
2. 运行 `npx cap open android`
3. 点击构建或运行按钮

### 安装到Android手机

```bash
# 通过ADB安装
adb install android/app/build/outputs/apk/debug/app-debug.apk
```

或传输APK文件到手机，点击安装。

## iOS 版本

**注意：** iOS开发只能在macOS上进行

### 方法一：使用Xcode（推荐）

#### 前置要求
- macOS系统
- Xcode（从App Store安装）
- CocoaPods (`sudo gem install cocoapods`)

#### 构建步骤

```bash
cd /home/admin/workspace/plant-care-assistant

# 1. 安装依赖
npm install

# 2. 构建Web资源
npm run build

# 3. 同步iOS项目
npx cap sync ios

# 4. 打开Xcode
npx cap open ios

# 5. 在Xcode中
# - 选择目标设备（模拟器或真机）
# - 点击运行按钮（▶️）
```

### 方法二：使用命令行脚本

```bash
./build-ios.sh debug
```

### 安装到iPhone

#### 模拟器
- 直接在Xcode中选择模拟器运行

#### 真机（需要Apple ID）
1. 连接iPhone到Mac
2. 在Xcode中选择你的设备
3. 配置签名（Signing & Capabilities → 选择Apple ID）
4. 点击运行
5. 在iPhone上信任开发者（设置 → 通用 → VPN与设备管理）

### 发布到App Store

1. 注册Apple Developer账号（$99/年）
2. 在Xcode中配置发布签名
3. Product → Archive
4. 使用App Store Connect上传

详细步骤见 [`iOS打包指南.md`](./iOS打包指南.md)

## 常见问题

### Android

**Q: 安装时提示"解析包时出现问题"**
- 重新下载/构建APK
- 确保文件完整

**Q: 安装时提示"未知来源"**
- 设置 → 安全 → 允许未知来源安装

**Q: 应用打开后白屏**
- 确保 `npm run build` 成功
- 运行 `npx cap sync android`

### iOS

**Q: 构建失败 "No such module 'Capacitor'"**
```bash
cd ios/App
pod install
```

**Q: 真机安装后无法打开**
- 在iPhone上信任开发者证书
- 设置 → 通用 → VPN与设备管理

**Q: 需要Apple Developer账号吗？**
- 模拟器：不需要
- 真机测试：免费Apple ID即可（需信任证书）
- App Store发布：需要付费账号（$99/年）
