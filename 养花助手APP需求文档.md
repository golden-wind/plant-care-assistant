## 1. 产品概述

### 1.1 产品定位
养花助手是一款面向家庭绿植养护爱好者的智能提醒应用，通过整合植物养护知识库与实时天气数据，为用户提供个性化的植物养护建议和自动化提醒服务。

### 1.2 核心价值主张
- **降低养护门槛**：让没有经验的新手也能轻松养好花
- **智能化提醒**：结合天气数据，提供精准的浇水、施肥、防护建议
- **知识赋能**：建立完整的家庭常见植物养护知识库

### 1.3 目标用户
| 用户类型 | 特征描述 | 核心痛点 |
|---------|---------|---------|
| 新手养花者 | 刚开始养花，缺乏经验 | 不知道每种花的养护方法，容易养死 |
| 忙碌上班族 | 工作繁忙，经常忘记照料 | 忘记浇水、施肥的时间 |
| 多植物养护者 | 家里养多种植物 | 不同植物养护周期不同，难以管理 |
| 季节性养护者 | 根据季节更换植物 | 不了解不同季节的养护要点 |

### 2.2 功能详细说明

#### 2.2.1 植物管理模块

**添加植物**
- 支持从植物库中选择添加
- 支持拍照识别植物（扩展功能）
- 设置植物昵称、购买日期、当前状态
- 记录植物位置（室内/阳台/室外）

**我的花园**
- 网格/列表视图展示所有植物
- 按状态筛选（健康/需关注/异常）
- 快速操作入口（标记浇水/施肥）
- 植物卡片显示下次浇水时间

**植物档案**
- 基础信息：名称、科属、原产地
- 养护参数：浇水频率、光照需求、适宜温度、湿度要求
- 当前状态：上次浇水、上次施肥、生长阶段
- 成长相册：时间轴记录植物变化

#### 2.2.2 智能提醒模块

**浇水提醒**
- 基于植物种类默认浇水周期
- 结合天气数据智能调整（雨天延后、高温天提前）
- 支持自定义浇水频率
- 推送通知 + 应用内提醒

**施肥提醒**
- 按植物类型设置施肥周期（观叶/开花/多肉等）
- 季节调整（生长期增加、休眠期减少）
- 推荐肥料类型

**天气灾害预警**
| 天气情况 | 触发条件 | 提醒内容 |
|---------|---------|---------|
| 大风预警 | 风力≥6级 | 建议将花盆移至室内或避风处 |
| 暴雨预警 | 降雨量≥50mm | 建议将花盆移至避雨处，避免积水 |
| 高温预警 | 温度≥35°C | 建议增加浇水频率，避免暴晒 |
| 寒潮预警 | 温度≤5°C | 建议将畏寒植物移至室内保暖 |
| 干旱预警 | 连续7天无雨 | 提醒检查土壤湿度，及时补水 |

**养护日历**
- 月视图展示所有养护任务
- 支持任务完成打卡
- 历史养护记录查询

#### 2.2.3 知识库模块

**植物百科**
- 收录常见家庭绿植 200+ 种
- 每种植物包含：
  - 基本信息（名称、别名、科属、原产地）
  - 形态特征
  - 养护要点（光照、温度、水分、土壤、施肥）
  - 繁殖方法
  - 常见问题及解决方案

**养护指南分类**
| 分类维度 | 具体类别 |
|---------|---------|
| 按养护难度 | 新手友好、进阶挑战、专家级 |
| 按空间位置 | 阳台花卉、室内绿植、庭院植物 |
| 按功能用途 | 净化空气、观赏花卉、香草植物、多肉植物 |
| 按光照需求 | 喜阳植物、耐阴植物、散射光植物 |

**问题诊断**
- 症状选择器（叶片发黄/枯萎/虫害等）
- 智能匹配可能原因
- 提供解决方案建议

#### 2.2.4 天气集成模块

**实时天气**
- 当前温度、湿度、天气状况
- 24小时天气变化趋势
- 未来7天天气预报

**位置服务**
- GPS自动定位
- 支持手动选择城市
- 多地点管理（家/公司/父母家）

## 4. 用户流程设计

### 4.1 新用户引导流程

```
启动APP → 欢迎页 → 位置授权 → 选择城市 
   ↓
添加第一株植物 → 从植物库选择 → 设置昵称和位置
   ↓
设置养护提醒 → 完成引导 → 进入首页
```

### 4.2 日常核心流程

**早间查看流程**
```
打开APP → 首页今日任务 → 查看待浇水植物
   ↓
完成任务 → 打卡记录 → 查看天气提醒
```

**添加新植物流程**
```
点击"+" → 浏览/搜索植物 → 选择植物
   ↓
设置信息 → 确认添加 → 生成养护计划
```

### 4.3 信息架构

```
底部导航栏
```
